About this app

This Shiny app simplifies the labeling and rating process. Users can:

  • Upload data files in .txt, .csv, or .xlsx format.
  • Filter the dataset to only the rows that need labels or ratings.
  • Specify constraints on the labels or ratings to be entered (e.g. only numbers within a certain range).
  • View each item and enter a label/rating one at a time in a separate window. Each rating is automatically saved in the appropriate row of the dataframe.
  • Add a note for each rating when needed.
  • Download a new data file with new columns for ratings and any notes.

Credits
